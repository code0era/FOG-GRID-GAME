## Overview
FOG-GRID-GAME is an interactive React-based web game where players navigate a grid.
Players must avoid dynamically generated patterns of enemies.
The game features a visually stunning neon galaxy theme.
## Setup
To run the game locally, you need Node.js installed.
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
## Game Rules
- You start at the top-left corner of the grid.
- Your goal is to reach the bottom-right corner.
- Red squares signify enemies that move in specific patterns.
- If you hit an enemy, your health decreases.
- The game is won when you reach the end point alive.
## Technology Stack
- Framework: React 18
- Build Tool: Vite
- Styling: Vanilla CSS
